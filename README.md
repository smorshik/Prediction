# Titanic Fare Predictor
Проект по машинному обучению, который предсказывает стоимость билета на "Титанике" на основе данных о пассажире.

### О проекте
Я использую **линейную регрессию** для анализа связей между классом каюты, возрастом, полом пассажира и ценой его билета. Это учебный проект в рамках моего обучения в ВНТУ.

### Стек технологий
* **Python 3.10+**
* **Pandas** — обработка данных
* **Scikit-learn** — построение модели и скейлинг
* **Matplotlib** — визуализация результатов

### Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install pandas matplotlib scikit-learn`
3. Запустите файл: `python main.py`

### Результаты
Модель визуализирует разницу между реальными данными и предсказаниями, а также позволяет ввести данные своего "пассажира" и узнать примерную стоимость его билета.
